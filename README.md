## 🧑‍💻 How to Run Locally

To run the project locally, follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/dsmooot/portfolio.git
```

2. Install deps:

```bash
yarn install
```

3. Run the project locally:

```bash
yarn dev
```
